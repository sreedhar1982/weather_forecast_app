Go to Server folder and install below dependencies
npm install express
npm install react
npm install react-router

Go to Client folder and install below & above dependencies
npm install gulp
npm install --global gulp
npm install browserify
npm install reactify
npm install vinyl-source-stream
npm install react-dom
npm install body-parser

Reference: https://semaphoreci.com/community/tutorials/testing-react-components-with-enzyme-and-mocha
For testing please install libraries s below
$ npm install --save babel-core babel-loader babel-preset-airbnb babel-preset-es2015 babel-preset-react babel-preset-stage-0

Go to Server folder and run below command
npm server

Go to Client folder and run below command
gulp

Open the url http://localhost:8080 in Browser
